# React redux seed

##Infrastructure
install node.js and npm - [how to](https://docs.npmjs.com/getting-started/installing-node)

## Installation

```bash
$ npm install
```

### Run localy
```bash
$ npm start
```
Server will be started on [http://localhost:8080/](http://localhost:8080/)

### Production build
```bash
$ npm run build
```
Compiled files will be located at ./dist/ folder
